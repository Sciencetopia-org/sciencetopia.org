## 目标
做一个系统、完备、面向群众、永久免费的科学平台。
## 方案
以图结构的数据库嵌入海量互联网知识资源，以关联学习和路径计算 AI 作为核心程序，根据用户需要智能提供学习路径和所有相关资源。并提供前沿动态推送和交流、讨论、组织合作的空间。
